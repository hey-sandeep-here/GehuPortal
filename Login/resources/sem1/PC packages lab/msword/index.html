Some Concepts

Justify text

  • When you justify text, space is added between words so that both edges of each line are aligned wih both margins.
  • The last line in the paragraph is aligned left.

Align Text - left/ right

  • Text alignment is a paragraph formatting attribute that determines the appearance of the text in a whole paragraph.

Superscript and subscript

  • Superscript shortcut → Ctrl + shift + (plus sign +)
  • Subscript shortcut → Ctrl + (plus sign +)

Macros

  • To save sequences of commands or operating instruction, which can then be reused by self defined key instructions.
  • It is used to reduce repeatative tasks and to automate work steps.
  • Example → If we have to insert an image, table or logo repeatatively so, instead for making them from scratch we can record the steps and store them and give it a shortcut key to reuse them.

Mail Merge

  • Mail merge lets you create a batch of documents that are personalized for each recipient.
  • Use mail merge for bulk email, letters, labels and envelopes.
