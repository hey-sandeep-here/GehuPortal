Some questions

Explain Microsoft PowerPoint

  • Power Point is a part of Microsoft Office and is used to create slides for the presentation.
  • It contains slideshow presentation.
  • We can display the content more attractively by combining graphics, text etc.

What is the advantages of Microsoft PowerPoint

  • Microsoft Power Point helps the organizations to organize the content in visual form like adding charts, objects, poster and visuals.

Is it possible to do animation for objects in PowerPoint?

  • Yes, we can animate the pictures, tables, shapes, text and other objects.
  • On an object we can make effects to appear, disappear or move.

Why do we need to use animations in PowerPoint

  • To enhance the appearance of your slides
  • To make a powerpoint presentation more dynamic
  • We can also add sound to increase the power of your animation effects
